## 1. 采购策略总览

### 1.1 按阶段采购策略

| 阶段 | 数量 | 采购模式 | 供应商类型 | 周期要求 |
|---|---|---|---|---|
| 原型验证 (Phase 1-3) | 1–5 台 | 零售/小批 | 立创 LCSC / Mouser / Digikey / 淘宝 | 1–2 周 |
| 小批量试产 | 50–200 台 | 小批量直采 | LCSC / 1688 / 代理商 | 4–6 周 |
| 量产 (首批) | 200–500 台 | 批量直采 + 期货 | 原厂代理 (Arrow / Avnet / 世强 / 中电) | 8–12 周 |
| 稳定量产 | >500 台/年 | 期货 + VMI | 原厂代理 + 现货商备货 | 按季度排产 |

### 1.2 供应商分级

| 级别 | 类型 | 代表供应商 | 适用场景 |
|---|---|---|---|
| **T1** 原厂/一级代理 | 原厂授权分销商 | Arrow, Avnet, Future, Digikey, Mouser | 关键 IC (TI, ST) |
| **T2** 国内授权代理 | 国内授权渠道 | 中电港 (CECport), 世强 (Sekorm), 力源 (P&S) | 批量采购 TI / onsemi |
| **T3** 现货/平台 | 现货分销平台 | LCSC (立创), 1688, 淘宝 | 阻容/通用件/小批量 |
| **T4** 国产替代渠道 | 直接联系原厂 | 圣邦微 (SGM), 迈诺斯 (MINOS), 开步睿思 | 国产化降本 |

## 2. 关键 IC 供应评估

### 2.1 MOSFET — M90N20

| 参数 | 信息 |
|---|---|
| **品牌** | MINOS (迈诺斯) — 台系/国产, IDM 模式 |
| **替代方案** | IRFP260NPbF (Infineon), IXFN90N20 (IXYS), NTHL090N65S3H (onsemi) |
| **MOQ** | 50–100 (LCSC) / 500 (原厂) |
| **单价** | ¥8–12/颗 (批量 100+) |
| **交期 (LT)** | 4–8 周 (原厂) / 现货 (LCSC) |
| **供应风险** | 低 — 多品牌兼容, TO-247 封装标准件 |
| **建议动作** | 原型用 LCSC 现货, 量产前签 MINOS 代理协议 |

### 2.2 MCU — C2000 F28E12x (或 F280013x)

| 参数 | 信息 |
|---|---|
| **品牌** | Texas Instruments (TI) |
| **替代方案** | STM32G4 (HRTIM) — 需重构代码; AT32F435 (雅特力) — 无 HRPWM |
| **MOQ** | 160–250 (代理), 1 (Mouser/LCSC) |
| **单价** | ¥60–80 (F28E12x) / ¥35–50 (F280013x) |
| **交期 (LT)** | 8–12 周 (代理期货) / 现货 (Mouser 有货) |
| **供应风险** | **中高** — TI 在 2021-2023 有过严重缺货, 目前好转但仍需提前锁定 |
| **建议动作** | 原型: Mouser / Digikey 现货; 小批量: 立即从 Arrow / Avnet 下期货单 |

| 物料 | 风险等级 | 风险描述 | 缓解措施 |

| M90N20 (MINOS) | 🟡 中 | 国内品牌, 渠道不成熟 | 锁定现货; 备选 IRFP260N / IXFN90N20 |
| 反激变压器 (定制) | 🟡 中 | 定制件, 打样周期长 | 提前启动; 标准磁芯+绕线厂 3 家备选 |

| ADS1115 (TI) | 🟢 低 | 通用料, 多来源 | MCP3421 替代 |

**风险等级**: 🔴 高 = 预计需 >1 月前置期或唯一来源 | 🟡 中 = 需关注交期 | 🟢 低 = 充足供应
