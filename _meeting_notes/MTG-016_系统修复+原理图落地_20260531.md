## 修复清单

### 1. 功率级原理图
- **问题**: 工作树被gen_sch.py覆盖清空（23行骨架→1648行内容丢失）
- **修复**: `git checkout HEAD -- EL-1000_PowerStage.kicad_sch` 恢复
- **验证**: kicad-cli可解析输出XML，含54个符号

### 2. 控制电路 KiCad 原理图 (NEW)
- **输入**: `SYS-SCH-002_控制电路原理图设计说明_v0.1.docx`
- **输出**: `EL-1000_ControlCircuit.kicad_sch` (57KB, 92元件)
- **内容**: C2000最小系统(晶振/JTAG/复位/退耦)、ESP32-S3模组、SGM71622R2 DAC、ADS1115 ADC、ISO7741隔离、ADuM1201/3160隔离、AMS1117 LDO、LCD连接器、编码器、按键矩阵、看门狗TPS3823
- **生成脚本**: `gen_ctrl_sch.py`

### 3. 电源树 KiCad 原理图 (NEW)
- **输入**: `SYS-SCH-003_电源树原理图设计说明_v0.1.docx`
- **输出**: `EL-1000_PowerTree.kicad_sch` (60KB, 65元件)
- **内容**: AC输入(EMI+NTC+泄放)、整流+母线电容、反激UCC28C45(COMP/CS/RTCT/OUT/VCC/VREF)、Buck1 +15V/1A(IRF9540)、Buck2 +12V/2A(自举)、Buck3 +5V/3A(自举)、LDO AMS1117-3.3、-15V电荷泵
- **生成脚本**: `gen_psu_sch.py`
- ⚠️ **注意**: 需在KiCad GUI中完善实际连线（脚本生成wire为示意性）

### 4. 控制算法设计文档 + PID仿真
- **文档**: `06_软件研发/docs/SYS-CTRL-001_控制算法设计说明_v0.1.docx.md`
- **仿真脚本**: `06_软件研发/firmware/scripts/pid_sim.py`
- **结果**: 5张分析图(阶跃响应/放大/Bode/Nyquist/Ki扫描)
- **关键发现**: 12×M90N20 跨导 ~120S, 相位裕度 109°, 闭环带宽 ~112Hz
- **发现的问题**: 固件中 max_integral=0.5 太保守，需≥0.95

### 5. 固件骨架 (39文件)
- **路径**: `06_软件研发/firmware/`
- **结构**: c2000/ (main+drivers+control+tasks) + esp32/ (main+components) + common/ + docs/
- **关键文件**: pid.c(反饱和积分的PID), hrpwm.c(25kHz HRPWM+MEP), adc.c(13ch扫描), cmpss.c(硬件比较器)
- ⚠️ **缺失4个文件**: protocol.c, wifi.h, wifi.c, data_log.c（迭代限制）

### 6. 安规认证策略
- **文档**: `07_测试验证/SYS-QA-001_安规认证策略_v0.1.md`
- **关键结论**: 2层策略——初期CE自声明+FCC SDoC(~¥27-45k), 量产全认证(~¥88-155k)
- **建议**: PCB布局阶段按EMI设计规范做，减后续认证返工

### 7. 整机BOM成本核算
- **文档**: `05_硬件研发/SYS-BOM-001_整机BOM成本核算_v0.1.md`
- **原有合计**: 功率级¥238 + 控制¥196 + 电源¥148 + 安全¥78 + 散热¥216 = ¥876
- **新增项**: 外壳¥175 + 连接器¥51 + PCB¥75 + 组装¥130 + 包装¥31 + 附件¥16 = ¥478
- **合计**: **¥1,354** (占零售¥3,999-5,999的22-34%, 毛利充足)
- **批量路径**: 200台→¥1,143, 500台→¥1,054

### 8. 供应链摸底
- **文档**: `08_供应链/SYS-SUP-001_供应链摸底框架_v0.1.md`
- **关键发现**: C2000 F28E12x🔴高风险(8-12周), ESP32-S3🟢低风险(2-4周)
- **备选方案**: STM32G4系列作为C2000替代

### 9. 快速原型验证策略
- **文档**: `07_测试验证/SYS-TEST-001_快速原型验证策略_v0.1.md`
