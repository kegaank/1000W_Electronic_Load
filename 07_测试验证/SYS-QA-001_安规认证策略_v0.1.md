## 1. 适用标准

EL-1000 为可编程直流电子负载，归类为 **实验室电气设备 / 测量、控制和实验室用电气设备**，主要适用标准如下：

### 1.1 安全标准

| 标准 / 法规 | 适用区域 | 说明 |
|---|---|---|
| **IEC 61010-1** | 国际 | 测量、控制和实验室用电气设备的安全要求 — 第1部分：通用要求 |
| **EN 61010-1** | 欧盟 | IEC 61010-1 的欧盟采纳版，CE 标志核心安全标准 |
| **UL 61010-1** | 美国/北美 | UL 采纳版，与 IEC 61010-1 技术等同，但 NRTL 认证需 UL 版本 |
| **GB 4793.1** | 中国 | 对应 IEC 61010-1 的国标，CCC 认证基础标准 |

> EL-1000 工作电压 **< 600V DC**，额定功率 **1000W**，属于 **I 类测量设备**（CAT I / II 环境），适用过电压类别 II，污染等级 2。

### 1.2 EMC 标准

| 标准 / 法规 | 适用区域 | 说明 |
|---|---|---|
| **EN 55032 (CISPR 32)** | 欧盟 | 多媒体设备辐射 & 传导发射限值 |
| **EN 55035 (CISPR 35)** | 欧盟 | 多媒体设备抗扰度要求 |
| **EN 61000-3-2** | 欧盟 | 谐波电流发射限值（≤16A/相） |
| **EN 61000-3-3** | 欧盟 | 电压波动和闪烁限值 |
| **FCC Part 15 Subpart B** | 美国 | 无意辐射体发射限值（Class A / Class B） |
| **GB/T 9254.1** | 中国 | 对应 EN 55032 的国标 |

### 1.3 环保 / 能效

| 标准 / 法规 | 适用区域 | 说明 |
|---|---|---|
| **RoHS (EU 2015/863)** | 欧盟 | 有害物质限制（10项） |
| **WEEE (EU 2012/19)** | 欧盟 | 废弃电气电子设备指令 |
| **REACH (EU 1907/2006)** | 欧盟 | 化学品注册、评估、授权与限制 |
| **China RoHS (GB/T 26572)** | 中国 | 电子电器产品有害物质限制 |

## 2. 认证路线选择

### 2.1 策略建议

| 目标市场 | 认证方案 | 预计费用 (¥) | 周期 | 说明 |
|---|---|---|---|---|
| **欧盟 (EU)** | **CE (EMC + LVD)** | 40,000–70,000 | 2–3 个月 | 自声明 (DoC) + 技术文档 + 测试报告 |
| **美国 (USA)** | **FCC SDoC** | 20,000–40,000 | 1–2 个月 | 供应商符合性声明，无需 FCC 发证 |
| **美国 (USA)** | **FCC + NRTL (UL)** | 80,000–150,000 | 4–6 个月 | 需 UL 授权 NRTL 测试，费用较高 |
| **中国 (CN)** | **CCC 自愿认证** | 30,000–50,000 | 3–4 个月 | 非强制性，但渠道商可能要求 |
| **全球通用** | **CB 报告** | 30,000–50,000 | 3–4 个月 | IECEE CB 体系，一国测试多国认可 |

### 2.2 推荐路线（按阶段）

#### 阶段一：原型验证 → 小批量（首批 50–200 台）
- **CE 自声明** (EMC + LVD)：内部预扫 + 第三方 EMC 预测试
- **FCC SDoC**：自行测试并出具报告
- **RoHS / REACH 合规声明**
- **目标费用**: ~¥30,000–50,000（预测试 + 报告编制）

#### 阶段二：量产（批次 >500 台/年）
- **CE 全项认证** + **FCC 正式测试** + **CB 报告**（覆盖全球）
- **考虑 UL 认证**（北美渠道要求时启动）
- **目标费用**: ~¥80,000–150,000

## 6. 风险缓解策略

### 6.1 设计阶段 EMC 防护

| ✅ 必做项 | 说明 |
|---|---|
| **PCB 4 层板设计** | 完整地平面，降低回路电感 |
| **输入 EMI 滤波器** | 共模扼流圈 + X/Y 电容，预留 π 型滤波位置 |
| **栅极驱动回路最小化** | 驱动走线<10mm，推挽紧贴 MOSFET |
| **功率回路与信号回路隔离** | 功率地-信号地单点连接，星形接地 |
| **MOSFET 开关边沿缓控** | 栅极电阻 10Ω，控制 di/dt、dv/dt |
| **C2000 PWM 输出 RC 滤波** | 每个 PWM 输出串联 33Ω + 100pF 对地 |
| **风扇 PWM 调速滤波** | 风扇线缆屏蔽 + 磁珠滤波 |
| **输入端口保护** | TVS + 共模扼流圈 + 放电管 |
| **屏蔽设计预留** | 外壳密封导电衬垫位置预留 |
| **Layout 评审检查项** | 重点检查高速信号回路面积、晶振/时钟走线 |

### 6.2 安全设计

| ✅ 必做项 | 说明 |
|---|---|
| **保险丝 / 断路器** | 输入回路 - 变压器原边保险，功率级硬件 OCP |
| **输入/输出隔离** | 控制回路与功率回路通过隔离电源 + 数字隔离器 |
| **紧急关断 (E-Stop)** | 硬件独立关闭 MOSFET 栅极驱动 |
| **温度保护** | NTC 多点监测 + 固件 OTP + 硬件 OTP 冗余 |
| **外壳防护** | 接地端子 + 金属外壳接地 + 防手指触碰 |
| **爬电距离** | 确保初级-次级间 ≥5.5mm (双重绝缘) |
| **所有高压区域标示** | 警告标签 ≥ 高压 50V 区域 |

### 6.3 预扫安排

| 阶段 | 动作 | 供应商 |
|---|---|---|
| PCB 投板前 | 关键信号 SI/PI 仿真 (可选) | 内部 / Altium 仿真 |
| 第一版样机 (W6) | EMC 传导 + 辐射预扫 | 华测检测 (CTI) / 莱茵 (TÜV) |
| 整改后复测 (W8) | 传导 + 辐射验证预扫 | 同上 |
| 正式测试前 (W10) | 全项预扫 (ESD / EFT / Surge) | 同上 |

### 6.4 认证策略决策树

```
是否出口欧盟？
 ├─ 是 → 是否需要 CE 全项？
 │       ├─ 是 → 委托 TÜV/SGS 全项测试 + 技术文档
 │       └─ 否 → 自声明 DoC + 预测试报告
 ├─ 否 → 是否出口美国？
 │       ├─ 是 → FCC SDoC + 考虑 NRTL (UL)
 │       └─ 否 → 是否需要 CB 报告（全球覆盖）？
 │               ├─ 是 → CB 测试 + 转国家差异
 │               └─ 否 → 仅 RoHS 合规声明
```

## 9. 附录：CE 标志流程

```
1. 确定适用指令 (EMC + LVD)
2. 确定适用协调标准
3. 准备技术文档 (TCF)
4. 执行内部或委托测试
5. 编制符合性声明 (DoC)
6. 加贴 CE 标志
7. 保留技术文档备查 (至少 10 年)
8. 指定欧盟授权代表 (EEA 外制造商)
```
