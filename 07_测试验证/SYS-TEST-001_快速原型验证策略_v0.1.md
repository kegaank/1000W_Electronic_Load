**项目**: EL-1000 1000W 可编程直流电子负载  
**版本**: v0.1  
**日期**: 2026-05-31  
**状态**: 初稿

---

## 1. 概述

本文档定义 EL-1000 的三阶段快速原型验证策略，从单管验证到 12 管满功率，逐级降低技术风险。采用 C2000 LaunchPad 开发板 + 面包板/打样 PCB 的方式，在正式 PCB 投板前完成核心算法和功率级的可行性验证。

### 1.1 核心理念

```
┌─────────────────────────────────────────────────────────────┐
│                     快速迭代 · 分阶段验证                      │
│                                                             │
│   Phase 1 (W7-W8)           Phase 2 (W9-W11)     Phase 3    │
│   ┌───────────────┐      ┌───────────────┐    ┌───────────┐│
│   │ 单管 · 面包板  │ ──→ │ 4管 · 验证板   │ ──→│ 12管 · 整机││
│   │  100W  max     │      │  400W  max     │    │ 1000W max ││
│   │  HRPWM + ADC   │      │ 均流+热分布    │    │ 全面测试  ││
│   └───────────────┘      └───────────────┘    └───────────┘│
│       2 周                   3 周                  3 周     │
│                                                             │
│   Total: 8 周 (W7-W14)                                      │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 决策门 (Phase Gate)

| Phase | Exit Criteria (通过条件) | 失败回退策略 |
|---|---|---|
| Phase 1 → Phase 2 | HRPWM 输出正常, ADC 采样精度 <1%, PID 闭环稳定 | 排查 HRPWM 配置/ADC 噪声 |
| Phase 2 → Phase 3 | 4 管均流误差 <10%, 散热温差 <15°C, 无开关震荡 | 优化栅极驱动/均流电阻 |
| Phase 3 → 定型 | 1000W 满载稳定, 效率合格, 无热失控 | 调整散热/保护配置 |

## 2. Phase 1: 单管验证 (W7–W8)

### 2.1 目标

在 C2000 LaunchPad + 面包板上搭建 **单路 M90N20** 电子负载核心回路，验证：
- ✅ HRPWM 输出 100kHz–1MHz 可调
- ✅ ADC 高速采样 (≥20kSPS)
- ✅ PID 闭环恒流/恒压控制
- ✅ 读取并校准时序

### 2.2 硬件搭建

```

                     │         C2000 LaunchPad              │
                     │  (LAUNCHXL-F28E12x / F280013x)       │

                     │  GPIO0 ─→ (HRPWM) ─────────────┐    │
                     │  ADCIN0 ←─ (电流反馈) ←──┐     │    │
                     │  ADCIN1 ←─ (电压反馈) ←──┤     │    │

                                                │     │
```

### 2.4 测试方案

#### 2.4.1 HRPWM 输出验证

| 测试项 | 方法 | 判定标准 |
|---|---|---|
| PWM 频率 | 示波器测量 PWM 输出脚 | 设定值与实测值偏差 <1% |
| PWM 分辨率 | 逐步调整占空比 (0.1%步进) | 可分辨 <0.1% 步进变化 |
| 最小/最大占空比 | 设置 0% 和 100% | 0% 时 MOSFET 完全关断 |
| 死区时间 | 示波器查看互补 PWM (若使用) | 50–200ns 可配置 |

#### 2.4.2 ADC 采样验证

| 测试项 | 方法 | 判定标准 |
|---|---|---|
| 电流采样精度 | 串联标准电流表, 对比 ADC 读数 | 满量程误差 <1% |
| 电压采样精度 | 并联标准电压表, 对比 ADC 读数 | 满量程误差 <0.5% |
| 采样率 | GPIO 翻转法测采样周期 | ≥20kSPS (每个通道) |
| 噪声电平 | 短接 ADC 输入测量本底噪声 | 峰峰值 ≤2LSB |

#### 2.4.3 PID 闭环测试

| 测试项 | 方法 | 判定标准 |
|---|---|---|
| 恒流 (CC) 模式 | 设定 1A, 2A, 5A, 加载 10-50V | 稳定时间 <5ms, 过冲 <5% |
| 恒压 (CV) 模式 | 设定 10V, 20V, 50V, 拉载 | 稳定时间 <5ms |
| 恒功率 (CP) 模式 | 设定 10W, 20W, 50W | 功率误差 <3% |
| 恒阻 (CR) 模式 | 设定 10Ω, 50Ω, 100Ω | 电阻误差 <3% |
| 负载阶跃响应 | 1A → 5A 阶跃 | 过冲 <10%, 恢复时间 <2ms |
| 动态负载 (动态模式) | 1kHz 方波, 1A↔5A | 跟踪无振荡 |

#### 2.4.4 极限条件测试 (Phase 1 100W 上限)

| 测试项 | 条件 | 判定标准 |
|---|---|---|
| 100W 满载 | 10V × 10A / 50V × 2A | MOSFET 温度 <100°C |
| 短路保护 | 输出短路 | MOSFET 关断, 不损坏 |
| 过流保护 | 超过设定值 120% | 硬件/软件保护触发 |

### 2.5 关键风险 (Phase 1)

| 风险 | 概率 | 影响 | 缓解 |
|---|---|---|---|
| C2000 HRPWM 配置复杂 | 中 | 延迟 1 周 | 先参考 TI C2000ware 例程 |
| ADC 噪声过大影响 PID | 低 | 精度不达标 | 硬件 RC 滤波 + 软件均值滤波 |
| 面包板寄生电感导致震荡 | 高 | 波形畸变 | 缩短飞线, 功率回路尽量短 |
| PID 参数调整困难 | 中 | 闭环不稳定 | 先 Ziegler-Nichols 整定, 再 fine-tune |

## 3. Phase 2: 4 管验证 (W9–W11)

### 3.1 目标

在打样 PCB (2层/4层) 上搭建 **4 路 M90N20 并联** 电子负载，验证：
- ✅ 4 管电流均流 (误差 <10%)
- ✅ 热分布均匀 (温差 <15°C)
- ✅ 栅极驱动波形正常 (无振荡/串扰)
- ✅ 400W 满载稳定性
- ✅ CAN 通信正常

### 3.2 硬件搭建

**与 Phase 1 的主要差异**:
- 取消面包板，使用打样 PCB (2 层 / 4 层可选)
- 4×M90N20 共享同一散热器
- C2000 LaunchPad 通过排针接入 PCB
- 增加电流采样调理电路 (差分放大器)
- 增加 CAN 通信接口

### 3.3 PCB 快速打样

| 周期 | 2–3 天 |
| 费用 | ¥30–50/片 × 5 片 = ¥150–250 |

### 3.5 测试方案

| 测试项 | 方法 | 判定标准 |
|---|---|---|
| 4 管静态均流 | 在 100W-400W 各功率点, 测量每管电流 | 电流偏差 ≤10% |
| 动态均流 | 负载切换瞬间 (50%→100%) | 偏差 ≤15% |
| 栅极驱动波形 | 4 路同时测量 (示波器 4 通道) | 上升/下降 <100ns, 无振铃 |
| 热分布 | 热像仪 / 热电偶, 400W 30 分钟 | 温差 ≤15°C, 最高 <100°C |
| 交叉导通测试 | 一路先开, 延迟后开另一路 | 无瞬时短路 |
| CAN 通信 | 发送/接收 1000 帧 | 误码率 <0.1% |
| 效率/功耗 | 测量 12V 辅助电源电流 | 辅助电源功耗 <5W |
| 400W 满载稳定性 | 连续运行 1 小时, 每 5 分钟记录 | 波动 <3%, 无过热关断 |

### 3.6 关键风险 (Phase 2)

| 风险 | 概率 | 影响 | 缓解 |
|---|---|---|---|
| 4 管均流不均 | 中 | 单管过载 | 增大检流电阻精度; 软件逐周期限流 |
| 栅极串扰 (米勒效应) | 中 | 误开通/关断 | 米勒钳位电路; 增大 Rgs |
| 散热不足 | 低 | 超温 | 确认散热器热阻 <0.5°C/W |
| 回路寄生电感大 | 中 | MOSFET 尖峰电压 | 叠层母线; 吸收电容 (snubber) |

- ✅ 12 管并联满功率 1000W
- ✅ 所有工作模式验证 (CC/CV/CP/CR/动态)
- ✅ 保护功能完整 (OCP/OVP/OTP/反接)
- ✅ OLED + ESP32 通信正常
- ✅ 热稳定性验证

- 完整 4 层 PCB (功率板+控制板 或 一体化设计)

- OLED 显示 + 旋钮/按键人机交互
- 完整保护电路 (硬件 OCP / OVP / OTP)

| Kp | 0.1–0.5 | 0.5–2.0 | 0.2–1.0 |
| Ki | 10–50 | 50–200 | 20–100 |
| Kd | 0 (先不用) | 0 (先不用) | 0 (先不用) |
| PWM 频率 | 100kHz | 100kHz | 100kHz |
| 采样率 | 20–50kSPS | 20–50kSPS | 20–50kSPS |

> 建议先用 Ziegler-Nichols 整定法：增大 Kp 直到振荡，记录 Ku 和 Tu，再计算 PID 参数。
