# SYS-BOM-001 整机BOM成本核算 v0.1

**状态**: 初稿

---

## 1. 概述

本文档汇总 EL-1000 整机物料清单 (BOM) 成本核算。基于前期分模块 BOM 汇总，补充机箱、连接器、PCB、组装、包装及配件等缺失项，形成完整整机成本基线。

**核算前提**:
- 数量: 小批量 50–200 台
- 单价: 以 LCSC / 淘宝 / 1688 批量价为准
- 币种: 人民币 (¥)
- 不含税、不含模具摊销
- 不含运费、关税

## 6. 风险 & 注意事项

1. **DAC 价格待确认** — SGM71622R2 (圣邦微) 尚未获得正式报价，暂估 ¥15–25/颗
2. **C2000 F28E12x 供应风险** — TI 芯片近期周期较长 (8–12 周)，需提前下单锁定
3. **AlN 垫片备选** — 若氮化铝垫片成本过高，可考虑 氧化铝 (Al₂O₃) 或 导热硅胶垫片 (TGP)，成本从 ¥36 降至 ¥12–18，但热阻增加 0.3–0.5°C/W
4. **运费未计入** — 各芯片运费分摊约 ¥5–10/台 (多供应商)
5. **不良率预留** — 生产不良率 3%–5%，约 ¥40–68/台 隐形成本
6. **价格波动** — 功率 MOSFET (M90N20) 价格受原材料 (硅/铜) 价格波动影响
