# ID-SPEC-001 EL-1000 电子负载 ID 设计需求说明书 v1.0

> 本文件为 1000W 可编程直流电子负载（项目代号 EL-1000）的工业设计需求文档，
> 供后续 3D 建模（Fusion 360 / SolidWorks）及手板制作使用。
> 对应阶段：04_外观结构/ID_工业设计
> 状态：定稿 | 日期：2026-05-28

---

## 1. 设计风格定位

### 1.1 品牌参考
- **参考品牌**：ITECH（艾德克斯）8500 系列电子负载
- **风格关键词**：专业仪器、分区布局、深色沉稳、精密感

### 1.2 竞品差异目标
- 不像入门级单片机壳
- 不像 DIY 实验设备
- 定位中型台式精密仪器

## 2. CMF 规范

### 2.1 颜色

| 应用区域 | 色号 | 色值 | 备注 |
|---------|------|------|------|
| 机身主色 | RAL 7021 | 深灰黑 (#2b2b2b) | 哑光表面 |
| 前面板 | RAL 7021 | 深灰黑渐变 | 顶部略浅，底部略深 |
| 装饰条/品牌色 | — | 青绿 #00a884 | 顶部横贯条 + 品牌元素 |
| 旋钮/按键 | — | 深灰 (#333-#444) | 哑光，浅色字符 |
| 正极端子 | — | 红色 | 四线接入 |
| 负极端子 | — | 蓝色 | 四线接入 |
| 屏幕底色 | — | 深蓝黑 (#0a1628) | IPS 暗色主题 |
| 前面板丝印 | — | 浅灰 (#ddd-#bbb) | 哑光 |

### 2.2 材质
| 部件 | 材质 | 表面处理 |
|------|------|---------|
| 机身壳体 | 冷轧钢板 SPCC，厚 1.2mm~1.5mm | 静电粉末喷涂 RAL 7021 哑光 |
| 前面板 | 铝合金 5052 或 6063 | 喷砂 + 阳极氧化深灰，局部拉丝 |
| 后盖 | 冷轧钢板，厚 1.0mm | 静电粉末喷涂 |
| 散热器 | 铝型材 AA6063 | 本色喷砂 + 阳极氧化 |
| 旋钮 | 铝合金 + 硅胶防滑圈 | 阳极氧化深灰 |
| 按键 | POM 或 ABS | 哑光注塑 + 激光雕刻字符 |
| 脚垫 | 硅橡胶 | 黑色，防滑 |

### 2.3 表面处理
- 前面板不做拉丝（ITECH 风格为细磨砂质感）
- 所有棱边倒角 R1~R2
- 接缝均匀 ≤0.3mm
- 丝印字符：白/浅灰，耐磨，键帽不掉字

## 3. 整机尺寸

### 3.1 外形尺寸
| 项目 | 数值 |
|------|------|
| 宽度 | 310 mm |
| 深度 | 370 mm（含前面板旋钮和后面板端子凸出） |

| 重量估算 | 约 4.5~5.5 kg |

### 3.2 散热布局

- 内部风扇：后侧 80mm×80mm ×2 轴向风扇向后排风
- 散热器脊柱居中，8×M90N20 背靠背贴装（与方案A一致）

## 6. 内部结构约束（继承方案A）

### 6.1 热管理
- **散热器**：铝挤型材，居中布置，导热面朝向功率管
- **功率管**：8×M90N20，背靠背对称安装于散热器两侧
- **风扇**：后侧 80×80×25mm ×2，抽风式，PWM 温控
- **风流路径**：侧板进风 → 散热器鳍片 → 后侧风扇排出
- **发热源隔离**：功率区与控制系统（C2000主板、ESP32）保持间距

### 6.2 PCB 安装
- 功率主板：紧贴散热器安装，用导热硅脂+绝缘垫片
- C2000 DSP 控制板：功率板上方或独立支架安装
- ESP32-S3 通信板：远离功率区域
- 反激辅助电源：独立小板，靠近 IEC 输入

### 6.3 结构件
- 脊柱式钣金骨架（2mm SPCC 折弯）作为整机结构支撑
- 散热器同时也是结构件，与骨架刚性连接
- 前面板：铝合金 CNC 或压铸，通过 M3 螺钉固定于骨架
